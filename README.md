## How To Compile & run

1) Download ![ns-3-dev-git](https://github.com/nsnam/ns-3-dev-git) 
2) Put __wifi_topology.cc__ in __scratch__ file,
3) Run the following commands

```bash
./waf configure
./waf --run scratch/wifi_topology.cc
```
